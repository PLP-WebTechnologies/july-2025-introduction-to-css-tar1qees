Why Learn CSS?

CSS makes web pages look good. It handles colors, layouts, fonts, spacing, and so much more. Without CSS, every website would be plain text!

The Box Model

  1. Content: the text or image
  2. Padding: space between content and border
  3. Border: the edge around the element
  4. Margin: the space outside the border
